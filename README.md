# fiscalizacao-obras-pr-tce
📊 Painel de Fiscalização de Obras Públicas — Paraná

📌 Sobre o Projeto

Este painel foi desenvolvido para apoiar as atividades de fiscalização do Tribunal de Contas do Estado do Paraná (TCE-PR), oferecendo uma visão analítica e interativa sobre as obras públicas municipais executadas no estado entre 2012 e 2024.

O objetivo é facilitar o monitoramento de investimentos públicos, identificar padrões de contratação e apoiar a tomada de decisão com base em dados reais.
